Понятно о CorelDRAW
=======

Большой сбоник материалов о CorelDRAW, и всего что с ним связано, написанных с 2007 года.

## Примечания

* Некоторые изображения книги содержат анимацию _(*.gif)_ и могут проигрываться только в онлайн версии.
